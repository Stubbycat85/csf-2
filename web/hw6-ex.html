Problem 3: If, Elif, and Else Statements with Mutually Exclusive Conditions

Mutually exclusive conditions cannot be true at the same time. An example is from the number guessing game from lecture: either the guess is less than the secret number, greater than, or equal to. For mutually exclusive conditions, it is good style to use the elif statements in a chain, rather than creating new if statements.

Rewrite your number guessing game code in hw6.py to use one elif, one if, and one else statement.

    • The middle coordinate in the second tuple in the set.
