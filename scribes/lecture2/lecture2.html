Flow Control (if/else statements/loops)

A program is like a waterfall, starts from the top and works its way down to the bottom.

Start:

a = “hello world”

print a

print a

print a

print a

print a

print a

print a

print “done”

End:

If/else statements – Is a Boolean value. If the condition is true then execute the code. If its false go to the next part of the program.

Example:

if <condition> == T:

-----> <execute code>

if <condition> == F:

-----> <do not execute code>

The “:” says that its going into a new scope

The indents represent the scope, if the condition is true for the if/else statements go into the next scope. There can be scopes within scopes such as a nested statement. Also the indents should remain consistent throughout the program, you can use a certain amount of spaces or just hit tab.

Example:

if isDepressed:

------>        #dropout = True

else:

------>        if not thunderandlightning or classAt10:

------>------>        #opt 1

------>        else:

------>------>        #opt 2

While loops:

while <condition> == T:

------>        <execute code>

Example:

#this gives a value to the variable x so it can be used in the while loop to avoid getting an undefined #error

x = 0

#this says while x is less than 6, execute the code in the scope. Once x is no longer less than 6 it will #stop

while x < 6:

------>        print “Hello world”

        #this is increasing the value of the variable x until the statement becomes false to end it.

------>        x = x + 1

#this tells you the while loop has ended since it is outside of the while loops scope meaning the loop is #no longer true

print “done”
